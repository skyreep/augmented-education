Welcome to the computer hardware library!

Motherboard

A motherboard is one of the most essential parts of a computer system. It holds together many of the crucial components of a computer, including the central processing unit (CPU), memory and connectors for input and output devices.

HDD

A computer hard disk drive (HDD) is the mechanism that controls the positioning, reading and writing of the hard disk, which furnishes data storage.

SSD

A solid-state drive (SSD) is a solid-state storage device that uses integrated circuit assemblies as memory to store data persistently.

PC Case

A computer case, also known as a computer chassis, tower, system unit, CPU (when referring to the case as a whole rather than the processor), or cabinet, is the enclosure that contains most of the components of a computer (usually excluding the display, keyboard, and mouse).

CPU

A central processing unit (CPU), also called a central processor or main processor, is the electronic circuitry within a computer that carries out the instructions of a computer program by performing the basic arithmetic, logic, controlling, and input/output (I/O) operations specified by the instructions.

Power Supply

A power supply unit (or PSU) converts mains AC to low-voltage regulated DC power for the internal components of a computer. Modern personal computers universally use switched-mode power supplies. Some power supplies have a manual switch for selecting input voltage, while others automatically adapt to the mains voltage.
